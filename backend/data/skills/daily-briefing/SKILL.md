# Daily Briefing

description: Generate a daily briefing from memory and recent activity

## When to use
When the user asks for a briefing, summary of the day, or "what's happening today".

## Instructions
1. Read today's daily log from memory
2. Check MEMORY.md for upcoming deadlines or tasks
3. Summarize into a concise briefing with:
   - Key items from yesterday
   - Today's priorities
   - Any reminders or follow-ups

## Format
Keep it short. Use bullet points. Start with "Good [morning/afternoon], here's your briefing:"
